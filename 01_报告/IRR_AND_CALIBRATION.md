# IRR_AND_CALIBRATION.md — 裁判可靠性的诚实说明(M27 修正老评委指出的偷换概念)

> **老评委指出**:我们用"两个 LLM 裁判一致性 κ=0.806"冒充了"人工标注者间一致性 IRR",
> 承诺的 n≥60 + 第二人工标注者没做——用相邻指标回应了未解决的问题。**这个批评成立,本文诚实修正。**

## 1. 概念厘清(不再混用)
| 指标 | 是什么 | 本项目状态 |
|---|---|---|
| **人工标注者间 IRR**(human inter-rater) | 两个**独立人类**对同一批答案打分的一致性 | ❌ **未建立**:校准仅单标注者(作者本人)n=20;无第二独立人类标注者 |
| **LLM 裁判间一致性** | 两个**LLM 裁判**(Coder-14B / internLM)逐题打分的一致性 | ✅ κ=0.806(N=392);**这不是人工 IRR**,只反映两个 LLM 是否打分一致,不反映"对不对" |
| **裁判 vs 客观真值一致性** | LLM 裁判 vs **确定性 keyword 结构真值**(客观计算,非人类意见) | ✅ 见 §3,n=56,κ=0.941 |

**修正动作(已执行)**:全项目凡"IRR κ=0.806"已改标为"**两 LLM 裁判间一致性 κ=0.806(非人工标注者 IRR)**",不再用它冒充裁判可靠性的人工验证。

## 2. 诚实降级:人工 IRR 未建立(已知局限)
- 裁判校准集为**单标注者(作者本人)n=20**,**16/20 是 0/1 极端样本**,分级子集仅 4 条(分级 MAE 0.041)。
- **没有第二个独立人类标注者**,因此**人类-人类 IRR 无法建立**。这是评测可靠性的**已知局限**,如实写入 `CONFIDENCE.md` 边界与本文,不掩饰。
- 承诺的"扩到 n≥60 + 第二人工标注者"中,**第二人工标注者无法提供**(无第二位独立人类);n≥60 用客观标准替代(见 §3)。

## 3. 缓解证据(M36-37 重做):裁判 vs 独立客观锚,n=20 → n=224
> ⚠ **旧 κ=0.941(n=56)已撤回**:它曾被当"人工 IRR 替代"且与评测同源(见 HONEST_LIMITATIONS 局限①)。
> 本节用 M36-37 重做的版本替换,**不再把它当 IRR 替代**,并扩样本、报告分歧方向。脚本 `scripts/build_calibration_expand.py`。

人工标注稀缺,但依赖答案的对错是**客观可算**的。用与评测管线解耦的确定性 keyword/`ast` 真值作客观锚,
在**扩大集**上测我们依赖的裁判与它的一致性(裁判取**跨家族解耦再判 gen=Coder≠judge**,红线满足):

| 解耦 LLM 裁判 vs 独立客观锚 | 值 |
|---|---|
| 判定数 | **n=224**(4 配置 × 56;原 n=20) |
| Spearman ρ(裁判连续分, 确定性 recall) | **0.894**(p=2.5e-79) |
| 二值(≥0.5)一致 Cohen κ | **0.795**(一致 201/224=89.7%) |
| 分歧方向 | 裁判宽松 20 / 裁判严格 3;系统性偏差仅 **+0.004** |

→ 在 n=224 上,**LLM 裁判与机械确定性评分器 89.7% 一致**,分歧小且方向透明(裁判略偏宽松但系统偏差近零)。
**这条只证"裁判可靠地套用了评分标准、未额外注入幻觉宽松",不证评分标准独立于裁判**——
评分标准的独立性另由 `ast` 独立解析交叉核对(87%,§见 JUDGE_INDEPENDENT_VALIDATION)支撑。

## 4. 最终诚实定位(M36-37 更新)
- **裁判可靠性的真实状态**:① 人工 IRR **仍未建立**(单标注者 n=20,自主 agent 无第二独立人类,**承认做不到**);
  ② 裁判 **vs 独立客观锚 κ=0.795(n=224,gen≠judge)**——证裁判可靠套用标准,非证标准独立;
  ③ 两 LLM 裁判间一致性 κ=0.806(说明结论不依赖单一裁判,但**不等于**裁判正确)。
- **更关键的进展(墙一)**:**核心结论已不再依赖裁判可靠性**——数量级 RAG(det-recall 0.159→0.747,p=6.9e-12)
  与 humble 脏识别(20%→93%)用**零 LLM 确定性指标**直接复算成立。见 `JUDGE_INDEPENDENT_VALIDATION.md`。
- **不再用任何 κ 冒充人工 IRR。** 三指标各司其职;人工 IRR 真空诚实保留为已知局限。
